# SOP 06 - Copywriting

Implements: Stage 6.

## Purpose

Generate all site copy for this pool service client. Output populates brand-dna.copy.* at Stage 10.1.

## Inputs (per-client)

```
clients/[Client Name]/Pipeline Data/strategy/strategy.json
clients/[Client Name]/Pipeline Data/seo/keyword-plan.json
niche-playbook/copy-locks.json
niche-playbook/copywriting.md
niche-playbook/copy-blocklist-additions.md
niche-playbook/vocabulary.json
```

## Procedure

1. Run 05-copy-deck agent.
2. Apply `niche-playbook/copywriting.md` voice register throughout. Apply `copy-blocklist-additions.md` blocklist.
3. Write `copy.hero`: eyebrow, headline (from `heroH1Format` with client city), subheadline, imageAlt.
4. Write `copy.heroTrustChips` (3-5 chips, priority order from `quantified-trust-templates.md`).
5. Write `copy.trustClaims` (5 claims from `copy-locks.json trustStripClaims` adjusted per client certifications).
6. Write all section copy bundles: services, process, pricing (offers), whyChoose, reviews, gallery, serviceAreas, serviceAreaCard, faq, cta, founder, blog.
7. Write locked strings: buttonText, submitButton, formHeader, formSubtext, privacyLine, mobileCallLabel, availableNow, footerCta, copyright.
8. Apply `riskReversalLine` from copy-locks.json as `copy.process.badgeSubtext` if no client-specific version is available.
9. Write `Pipeline Data/copy/copy-deck.json`.

## Pass criteria

- `copy.hero.headline` contains the client's city name.
- `copy.heroTrustChips.length >= 3`.
- All `__REQUIRED__` sentinel fields in brand-dna are replaced in the copy-deck.
- No em-dashes, no AI-vocab blocklist terms, no banned pool-services phrases.

## When this stage halts

- City name is not confirmed (cannot write headline without city).
- Services list is empty (cannot write services copy).

## Niche-specific notes

The pool service register is "knowledgeable neighbor." See `niche-playbook/copywriting.md` for full guidance.

The most common copy mistake in this niche: writing about the company instead of writing about the homeowner's problem. Every headline should address an outcome ("Crystal clear pool") or a fear ("Pool you can finally count on") before naming the company.
