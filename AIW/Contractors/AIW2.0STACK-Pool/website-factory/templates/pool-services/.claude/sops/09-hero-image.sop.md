# SOP 09 - Hero Image

Implements: Stage 9.

## Purpose

Generate or select the hero pool photograph that fills the right panel of the homepage hero section (45% of viewport width at desktop). This stage runs before Stage 10.1 build.

## Inputs (per-client)

```
clients/[Client Name]/Pipeline Data/assets/manifest.json
clients/[Client Name]/Pipeline Data/brand/brand-dna.json
niche-playbook/hero-mood-mapping.json
niche-playbook/hero-composition.md
niche-playbook/photo-manifest.json
```

## Procedure

1. Check `manifest.json` for the `hero` category. If a qualified hero photo exists (min 1440x900, no green water, no equipment close-ups), use it. Skip generation. Go to Step 4.
2. If no qualified hero photo: run 08-hero-image agent to generate via Nano Banana / Gemini Image API.
3. Use the prompt template from `niche-playbook/hero-composition.md` (Photo Brief for Nano Banana section), substituting client city, state, and mood from `hero-mood-mapping.json`. Use regional override if client's state matches a regionalOverrides entry.
4. Resize to 2400x1600px and save to `clients/[Client Name]/[Client Name] Assets/hero/pool-hero.jpg`.
5. Copy to the built site's `public/hero/pool-hero.jpg` at build time (Stage 10.1 handles this).
6. Record in `Pipeline Data/assets/manifest.json` under the `hero` category.

## Pass criteria

- `hero/pool-hero.jpg` exists at minimum 1440x900.
- Image shows: clear blue pool water, no green water, no people in distress, no equipment-only close-ups.
- File size under 600KB after optimization.

## When this stage halts

- Nano Banana API is unavailable and no client hero photo exists: flag as `MANUAL-DROP-NEEDED`, proceed with build using placeholder, note in delivery report.

## Niche-specific notes

See `niche-playbook/hero-composition.md` for the full composition brief. The hero image for pool-services must convey one thing: a clean, sparkling, resort-quality residential pool. This is the first image the homeowner sees and it directly represents the outcome they are paying for.

Default mood: `golden_hour_warm`. Arizona/Nevada/Texas clients: `bright_midday_clean`.
