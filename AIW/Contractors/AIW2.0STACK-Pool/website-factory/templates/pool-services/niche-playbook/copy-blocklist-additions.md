# Pool Services: Copy Blocklist Additions

These terms are banned in pool-services copy in addition to the universal AI-vocab blocklist at `references/copy/ai-vocab-blocklist.md`.

## Industry Cliches to Avoid

| Banned phrase | Why | Alternative |
|---|---|---|
| "Sparkling results" | Overused in every pool company's marketing | "Crystal clear water" |
| "Pool paradise" | Cheesy | Describe the actual outcome |
| "Dive in" as a headline | Overused pun | Use only in the very last CTA if irresistible |
| "Making a splash" | Cliche pun | Avoid entirely |
| "Your pool dreams" | Abstract | "Your clean pool" |
| "Pool perfection" | Superlative | "Consistently clean" |
| "We love pools" | Irrelevant to buyer | Skip the sentiment |
| "Pool guys" | Informal, outdated | "Technicians" or "pool technicians" |
| "Chemical wizardry" | Gimmicky | "Water chemistry balancing" |

## Forbidden Claims (Compliance)

| Phrase | Issue |
|---|---|
| "#1 pool service in [city]" | Cannot be substantiated without evidence |
| "Cheapest pool service" | Invites race to bottom, wrong positioning |
| "Instant results" | No pool can be cleaned instantly |
| "Your pool in 30 minutes" | Unsubstantiated claim |

## Em-Dash Rule

Never use an em-dash in any copy, anywhere. This is a global rule and applies to every page, every component, every tooltip. Replace with:
- A period (split the sentence)
- A comma (if the sentence structure allows)
- A colon (for lists)
- Parentheses (for asides)
