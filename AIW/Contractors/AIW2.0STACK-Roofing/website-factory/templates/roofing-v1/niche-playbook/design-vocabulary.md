# Roofing Design Vocabulary

## Shape language

Angular. 4px button radius. 6px card radius. No pill buttons. No heavy drop shadows. Borders instead of shadows for card separation.

## Color application

- Dark navy (#0A0F1A) backgrounds: Hero, ReviewTiles, WhyUs, Footer, FinalCTA
- Light (#F8F9FA) backgrounds: EstimateForm, ProcessSteps, FAQAccordion, ContactPage content
- Teal (#0ABFBC) primary: CTAs, active states, accent text, step numbers
- Gold (#F59E0B) accent: Stars only

## Typography rules

- All headings: Barlow Condensed, Bold (700), UPPERCASE
- H1: 56-72px desktop, 40-48px mobile
- H2: 40-48px desktop, 32-40px mobile
- H3: 20-24px, uppercase
- Body: Inter, 400, 14-16px, line-height 1.6
- Overline labels (section eyebrow text): Inter, 600, 11-12px, uppercase, 0.12em letter-spacing, teal color

## Spacing

- Section vertical padding: 80px desktop, 56px mobile (use `py-section-gap-lg` utility)
- Card padding: 24px
- Grid gap: 20-24px between cards

## Iconography

Solid Heroicons (HeroIcons v2 solid). No outline icons in dark sections. Teal color for feature icons in dark sections.

## Photography

Real job site photos only. No stock. Overlay required on hero (rgba(10,15,26,0.55)). Before/after pairs labeled with city and service type.
