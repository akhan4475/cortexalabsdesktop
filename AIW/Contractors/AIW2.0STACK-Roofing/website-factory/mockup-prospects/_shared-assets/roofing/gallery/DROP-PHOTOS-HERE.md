# Roofing Stock Gallery

Drop before/after photos here. The mockup script copies them automatically
into the template on every run -- no manual work per prospect.

## Naming convention

Pair photos with matching numbers:

    before-1.jpg   after-1.jpg
    before-2.jpg   after-2.jpg
    before-3.jpg   after-3.jpg

Non-paired photos also work (just use any filename):

    project-1.jpg
    storm-job-dallas.jpg

## What shows in the mockup

The BeforeAfterGallery section renders these photos labeled as
"Local Roofing Project" with the prospect's city auto-filled.
Looks real. Homeowners can't tell it's stock.

## Recommended shots to source (royalty-free or your own)

- Dark asphalt shingle replacement (before: worn/missing shingles, after: clean new roof)
- Storm damage + repair (hail dents visible in before)
- Metal roofing install (for variety)
- Aerial neighborhood shot (after only is fine)

Minimum for a convincing mockup: 3 before/after pairs (6 photos total).
